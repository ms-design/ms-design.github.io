MS Design

基于微软设计语言的 Vue 组件库

# 安装

使用 npm 或 yarn 来安装 MS Design

# with npm
npm install ms-design -S

# with yarn
yarn add ms-design

# 使用

import Vue from 'vue';
import msd from 'ms-design';
import 'ms-design/lib/ms-design.css';

Vue.use(msd);

# 浏览器支持

支持IE10及以上的现代主流浏览器
